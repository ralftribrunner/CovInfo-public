# CovInfo-public

Android and iOS application about the latest covid-19 numbers in Hungary. 

Charts: 
 * new cases 
 * active cases 
 * new cases (5 days)
 * new recovered (5 days) 
 * new deaths (5 days)

Numbers:
 * daily new cases
 * daily recovered
 * daily recovered
 * active cases
 * all recovered
 * all deaths
